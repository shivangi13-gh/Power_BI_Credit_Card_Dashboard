# Credit Card Weekly Analytics Dashboard

## 1. Project Objective
To develop a comprehensive credit card weekly dashboard that provides real-time insights into key performance metrics and trends, enabling stakeholders to monitor and analyze credit card operations effectively.

## 3. Dashboard Overview
**3.1 Credit Card Transaction Report**

Gives a consolidated view of transaction performance and revenue drivers:
- Overall revenue, interest earned, transaction amount, and transaction count
- Quarterly revenue and transaction count trends
- Revenue split by card type and transaction mode
- Key spending categories contributing to revenue

![Credit Card Transaction Report](/Images/credit_card-transaction.JPG)

**3.2 Credit Card Customer Report**

Provides insight into customer demographics and behavior:
- Revenue trends by gender and age group
- Weekly revenue pattern comparison between male and female customers
- Revenue contribution by job type, income group, and education level
- Geographic contribution from top-performing states

![Credit Card Customer Report](/Images/credit_card-customer.JPG)

## 5. Conclusion
This project brings together weekly credit card data to track performance, customer behavior, and key business trends through an interactive dashboard. By combining well-defined KPIs, trend analysis, and clear visual storytelling, the dashboard provides a structured view of how revenue, transactions, and customer segments evolve over time. The outcome is a practical reporting solution designed to support regular monitoring and insight-driven analysis.
